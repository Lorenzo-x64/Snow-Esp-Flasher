❄ Snow Transceiver

Flash the Snow Transceiver to your ESP32.

Click install, pick the right serial port, and the latest firmware and dashboard are written to your board in under a minute. No drivers, no IDE.

Your browser doesn't support Web Serial. Open this page in Chrome, Edge, or Opera on desktop. Web Serial requires a secure context. Open this page over HTTPS.
Target: ESP32 WROOM (4 MB)
Includes firmware + dashboard files (LittleFS).

What you need

  • A uPesy ESP32 WROOM 32E (or any compatible ESP32 with 4 MB flash)
  • A USB-C cable connected to your computer
  • A Chromium-based browser (Chrome, Edge, Opera) — Firefox and Safari don't support Web Serial

How it works

  1. Plug your ESP32 into a USB port.
  2. Click Install firmware, then pick the serial port for your board.
  3. Choose Install when prompted. The flasher erases the chip and writes the firmware + LittleFS dashboard files.
  4. When the flash finishes, the board reboots and starts the Wi-Fi access point Snow Transceiver (password 1234567890).
  5. Connect to that network and open http://192.168.4.1 to use the dashboard.

Troubleshooting

  • If the install button is greyed out, switch to Chrome / Edge / Opera on desktop.
  • If no serial port appears, install the USB-to-UART driver for your board (CP2102 or CH340 depending on the model) and reload this page.
  • If the flash hangs at “connecting”, hold the BOOT button on the board, click install again, and release once it starts.
  • The dashboard requires the LittleFS partition — this flasher writes it for you automatically.
